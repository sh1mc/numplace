# numplace
数独を解いてくれます。

# コンパイル
```
$ g++ -std=c++11 main.cpp
```
みたいな感じで適当にやってください

# 使い方
標準入力に以下のような初期盤面のデータを入れます。空白は0として扱っています。
```
530070000
600195000
098000060
800060003
400803001
700020006
060000280
000419005
000080079
```
テキストデータで保存して
```
$ ./a.out < input.txt
```
のように実行すると、自動で標準入力に放り込んでくれるので便利だと思います。
